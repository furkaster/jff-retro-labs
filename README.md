### Артефакты золотой эры программирования

Репозиторий с исходным кодом для экспериментов в **QBasic**, **MS-DOS** и низкоуровневой логике.

* **Направление:** Ретро-графика, математическая визуализация и все такое.
* **Среда:** MS-DOS / QBasic 4.5 / 7.1.
* **Цель:** Исследование возможностей классического железа и софта.
